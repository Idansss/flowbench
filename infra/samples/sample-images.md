# Sample Images for Testing

## Image Studio Testing

For testing the Bulk Image Studio, you'll need sample images.

### Quick Test Images:

**Option 1: Use Your Own**
- Drag any JPG, PNG, or WebP files from your computer

**Option 2: Generate Test Images**
Use this online tool: https://placehold.co

Example URLs:
- https://placehold.co/1920x1080/png
- https://placehold.co/800x600/jpg  
- https://placehold.co/400x400/webp

**Option 3: Public Domain Images**
- Unsplash: https://unsplash.com
- Pexels: https://pexels.com
- Pixabay: https://pixabay.com
